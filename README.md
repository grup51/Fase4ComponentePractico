# Sistema de Gestión de Clientes - POO

## Descripción
Sistema orientado a objetos en Python que gestiona clientes, servicios y reservas sin base de datos.

## Características
- Abstracción, herencia, polimorfismo
- Encapsulación
- Manejo avanzado de excepciones
- Logs de errores
- Simulación de 10 casos

## Ejecución
```bash
python main.py
